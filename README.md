# 📊 SQL Skills & Expertise

I am proficient in writing complex SQL queries for data analysis and manipulation. Here’s a breakdown of the technical skills I apply to handle relational databases:

### 🛠️ Core Querying & Filtering
* **Data Retrieval:** Expert in using `SELECT`, `DISTINCT`, and Aliases.
* **Filtering:** Proficient in `WHERE` clauses using `AND`, `OR`, `IN`, `BETWEEN`, and `LIKE` for pattern matching.
* **Sorting:** Using `ORDER BY` (ASC/DESC) to organize data effectively.

### 📈 Data Aggregation & Grouping
* **Summarizing Data:** Skilled in Aggregate Functions like `COUNT()`, `SUM()`, `AVG()`, `MIN()`, and `MAX()`.
* **Categorical Analysis:** Using `GROUP BY` to generate insights per category and `HAVING` to filter aggregated results.

### 🔗 Advanced Relational Concepts
* **Table Joins:** Deep understanding of `INNER JOIN`, `LEFT JOIN`, `RIGHT JOIN`, and `FULL OUTER JOIN` to combine data from multiple sources.
* **Subqueries:** Ability to write nested queries for complex data retrieval.
* **CTEs (Common Table Expressions):** Using `WITH` clauses to write clean, readable, and maintainable code.

### ⚙️ Analytical Functions & Logic
* **Window Functions:** Experienced in `ROW_NUMBER()`, `RANK()`, and `DENSE_RANK()` using `OVER` and `PARTITION BY`.
* **Conditional Logic:** Implementing `CASE WHEN` statements to create custom categories and data labels.

### 🧹 Data Cleaning
* **Quality Assurance:** Handling missing values with `IS NULL` / `IS NOT NULL` and using functions like `COALESCE` or `NULLIF` to ensure data integrity.
